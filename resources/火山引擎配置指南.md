# 火山引擎 AI 服务配置指南

本文档介绍如何配置火山引擎的 AI 服务以使用智题坊的 AI 功能。

## 1. 注册火山引擎账号

1. 访问 [火山引擎官网](https://www.volcengine.com/)
2. 点击"立即注册"创建账号
3. 完成实名认证（部分功能需要）

## 2. 开通方舟大模型服务

1. 登录火山引擎控制台
2. 在产品列表中找到"火山方舟"或访问 [方舟大模型平台](https://www.volcengine.com/product/ark)
3. 点击"立即使用"开通服务
4. 选择合适的计费方式（有免费额度可供测试）

## 3. 创建 API Key

1. 进入方舟控制台
2. 点击左侧菜单"API Key 管理"
3. 点击"创建 API Key"
4. 设置 Key 名称并确认创建
5. **重要：** 复制并妥善保存 API Key（只显示一次）

## 4. 获取模型端点信息

1. 在方舟控制台中，点击"模型推理"
2. 选择您要使用的模型（如 Doubao-pro-4k、Doubao-lite-4k 等）
3. 记录以下信息：
   - **API Base URL**: `https://ark.cn-beijing.volces.com/api/v3`
   - **模型 ID**: 如 `ep-xxxxxxxxxx-xxxxx`（在模型详情页获取）

## 5. 在智题坊中配置

在系统设置页面填写以下信息：

| 配置项 | 值 |
|--------|-----|
| API Base URL | `https://ark.cn-beijing.volces.com/api/v3` |
| API Key | 您在步骤3中创建的 API Key |
| 模型名称 | 您的模型端点 ID（如 `ep-xxxxxxxxxx-xxxxx`） |
| 视觉模型 | 支持图像的模型端点 ID（可选） |

### 配置示例

```
API Base URL: https://ark.cn-beijing.volces.com/api/v3
API Key: ark-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
模型名称: ep-20240101120000-abcde
视觉模型: ep-20240101120000-vision
Temperature: 0.3
Max Tokens: 4096
```

## 6. 测试连接

1. 填写完配置后，点击"保存配置"
2. 点击"测试连接"按钮
3. 如果显示"连接成功"，则配置正确
4. 如果失败，请检查：
   - API Key 是否正确
   - 模型端点 ID 是否正确
   - 网络是否正常
   - 账户是否有足够额度

## 7. 常见问题

### Q: 提示"API Key无效"

A: 请检查 API Key 是否正确复制，确保没有多余的空格。

### Q: 提示"模型不存在"

A: 请确认模型端点 ID 正确，且该模型已在您的账户中启用。

### Q: 提示"配额不足"

A: 请在火山引擎控制台查看账户余额或免费额度是否已用完。

### Q: 图片识别功能不可用

A: 需要配置支持视觉功能的模型作为"视觉模型"。

## 8. 推荐模型

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| Doubao-pro-4k | 高性能，推理速度快 | 题目解析、智能出题 |
| Doubao-lite-4k | 成本低，响应快 | 简单的文本处理 |
| Doubao-vision | 支持图像理解 | 图片题目识别 |

## 9. 注意事项

- API Key 请妥善保管，不要泄露给他人
- 建议设置合理的 Token 限制以控制成本
- 火山引擎 API 遵循 OpenAI 兼容格式
- 如遇问题，可查看 [火山引擎文档](https://www.volcengine.com/docs/82379/1099475)
