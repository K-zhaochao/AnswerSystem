# 资源文件目录

此目录包含应用程序所需的资源文件：

## icons/

- `app.ico` - 应用程序图标（需要手动添加一个 256x256 的 ICO 文件）

## styles/

- `main.qss` - Qt 样式表文件

## 图标制作建议

1. 设计一个代表"答题系统"的图标，可以使用：
   - 书本/试卷图案
   - 勾选/打钩符号
   - 灯泡/智能符号

2. 推荐尺寸：256x256 像素
3. 格式：ICO 文件（包含多种尺寸 16x16, 32x32, 48x48, 256x256）
4. 可使用在线工具如 favicon.io 或 icoconvert.com 制作
